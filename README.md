# AWS Study Site（AWS認定 全般）

AWS認定試験向けの学習用サイトです。ドメイン別の学習項目（タスク/知識）を整理し、検索しながら復習できます。必要に応じて Gemini API を使って「用語の解説」や「模擬問題」を生成できます。

現在は試験切り替え機能の動作確認として **ANS / CLF** を公開しています（他の試験は雛形のみ）。

- 主要ファイル: [index.html](index.html)

## 使い方

1. 公開されているサイト（GitHub Pages）にアクセスします
2. 画面右上の **試験ボタン（ANS / CLF）** で試験を切り替えます
3. 画面上部のタブで **ドメイン** を切り替え、タスク一覧を確認します
4. 検索欄で、タスク名・日本語タイトル・知識項目を横断検索します

### AI 機能（任意）

Gemini API キーを設定すると、以下が使えます。

- 知識項目の「✨ 解説」
- タスクの「✨ 模擬問題を作成」

手順:

1. 画面の **API Key（⚙️）** を開く
2. **Google Gemini API Key** を入力して保存
3. 各項目の「✨」ボタンを押す

※ API キーはブラウザの LocalStorage に保存されます。共有PCなどではキーの削除を推奨します。

## 学習リソースの追加要望

追加してほしい学習リソースがあれば、GitHub の Issues を作成してください。

## フォークして自分用にカスタマイズ

このリポジトリは OSS として、フォークして独自の学習サイトに作り替えることも可能です。

- 試験データは `js/data/*.js` に分割されているので、フォーク後に自分用データへ差し替え/追加できます
- 公開する試験は `js/config.js` の `PUBLIC_EXAM_IDS` で制御できます
